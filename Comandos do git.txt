Referência De Comando Git.

Carregar Todos Os Arquivos Em Um Diretório Local Para Um Novo Repositório Git
Se Você Tiver Um Projeto No Seu Computador E Acabou De Criar Um Repositório Git Vazio No Github, Use Esses Comandos Para Fazer O Upload De Tudo No Git.

Cd Seu Diretório
Git Init
Git Remote Add Origin Git@Github.Com: Seu Nome De Usuário / Seu-Repo.Git
Git Add . 
Git Commit -Am " Mensagem "   
Git Push -U Origin Master
Baixe Todos Os Arquivos Do Repositório Git Para Um Diretório Local
O Oposto Da Opção Acima - Por Exemplo, Se O Seu Repositório Existir No Github E Você Estiver Trabalhando No Outro Computador Local. Execute Este Comando Para Onde Você Deseja Que O Novo Diretório Apareça (Não No Diretório Que Você Deseja Que Apareça).

Git Clone Git@Github.Com: Seu-Nome-De-Usuário / Seu-Repo.Git        # Usando Ssh   
Git Clone Https://Github.Com/Your-Username/Your-Repo.Git # Usando Https
Remover Um Arquivo Do Cache Do Git
Remova Um Arquivo Em Cache.

Git Rm -R —-Cached File.Txt
Substituir Diretório Local Inteiro
Se Você Tiver Alguns Conflitos De Mesclagem Ou Acidentalmente, Começou A Fazer Uma Alteração No Diretório Local Antes De Extrair Alterações Do Mestre, Veja Como Você Pode Reverter Ou Acessar O Diretório Local Para Quem Não Está No Github.

Git Fetch --All
Git Reset --Hard Origem / Master
Ignorar Um Diretório
Se Você Estiver Acompanhando Um Diretório E Depois Decidir Ignorar O Diretório Inteiro, Simplesmente Use-Lo A .Gitignorenão É Suficiente. Primeiro, Você Deve Adicionar O Diretório Ao .Gitignore E, Em Seguida, Executar Este Comando:

Git Rm -R --Cache Seu Diretório
Em Seguida, Pressione Como Alterações.

ADICIONE .GITIGNORE A UM REPOSITÓRIO EXISTENTE
SEMELHANTE AO ANTERIOR, MAS VOCÊ PODE SELECIONAR UM .GITIGNORECOM MUITAS ALTERAÇÕES.

Git Rm -R --Cached . 
Git Add .

Saindo Do Vim
PARA INICIANTES NA LINHA DE COMANDO, TERMINAR NA TELA DE MENSAGEM DE CONFIRMAÇÃO (NORMALMENTE QUANDO VOCÊ ESQUECE OU -MSINALIZAR UMA ADIÇÃO DE VERIFICAÇÃO ) É CONFUSO PORQUE PRESSIONAR ESCAPE (OU CTRL+ C) NÃO SAI DA TELA, POIS O EDITOR PADRÃO DO GIT É O VIM. EM VEZ DISSO, PRESSIONE ESCAPE (SE VOCÊ COMEÇAR A TENTAR DIGITAR ALGUMA COISA) E DIGITE O SEGUINTE COMANDO:

: Q
E Pressione Enter E Você Retornará Para Onde Estava.
